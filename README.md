# IoT-Based Solar Energy Monitoring & Analysis System

## Overview
This project monitors real-time solar voltage, current, and power output using 
Arduino and an ESP8266 Wi-Fi module. The system enables remote monitoring of 
solar panel performance through cloud connectivity.

## Hardware Components
- Arduino Uno
- Voltage Sensor Module
- Current Sensor
- ESP8266 Wi-Fi Module
- Solar Panel

## Software & Tools
- Embedded C (Arduino IDE)
- ESP8266 WiFi Library
- Serial Monitor for debugging
